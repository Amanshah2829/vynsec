# Vynsec Creations

A sleek, modern IT solutions platform for a global clientele.

## Features
- **Cybersecurity Focus**: Dedicated VAPT and Safety Training sections.
- **Custom Development**: Showcasing PWA, SaaS, ERP, and CMS expertise.
- **Global 24/7 Support**: Emphasis on worldwide availability.
- **Responsive Design**: Optimized for all devices with a premium cyber aesthetic.

## Tech Stack
- Next.js 15
- React 19
- Tailwind CSS
- Shadcn UI
- Lucide Icons

## Development
To start the development server:
```bash
npm run dev
```
